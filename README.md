# EV-Selection-DSS-CPP

Інтелектуальна система підбору електромобілів для клієнтів автосалону  
**Реалізація на C++ (Windows Forms)**

## Про проект

Це настільна Windows-застосунок, розроблений у середовищі Visual Studio (C++/CLI), який дозволяє клієнтам автосалону швидко підібрати оптимальний електромобіль за індивідуальними критеріями.

**Метод оцінки:** Weighted Sum Model (WSM)  
**Функціонал:**
- Завантаження даних з CSV-файлу (перетягування або вибір файлу)
- Інтерактивні фільтри за бюджетом, запасом ходу, потужністю зарядки, розгоном, кількістю місць тощо
- Налаштування вагових коефіцієнтів критеріїв
- Автоматичний розрахунок рейтингу (Score)
- Пояснення рекомендацій (explainability) для кожної моделі
- Сортування результатів за рейтингом

**Технології:**
- C++ / C++/CLI
- Windows Forms
- DataTable + DataGridView
- TrackBar, ComboBox, Drag & Drop
